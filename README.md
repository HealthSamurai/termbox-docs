# Termbox Documentation

This repository contains the documentation for [Termbox](https://www.health-samurai.io/termbox) - FHIR terminology server for ValueSets and CodeSystems.

## Structure

```
termbox-docs/
├── docs/           # Markdown documentation files
├── assets/         # Images and other assets
├── SUMMARY.md      # Navigation structure
└── redirects.yaml  # URL redirects
```

## Local Development

Documentation is rendered by the [health-samurai.io](https://health-samurai.io) portal.

## Contributing

1. Edit markdown files in `docs/`
2. Update `SUMMARY.md` if adding new pages
3. Submit a pull request
